About

Hexacon, formerly InfinityCraft, was a Minecraft Survival Multiplayer Server (SMP) that ran from 2018 to 2021. The server was a place for creators on youtube to come together to make videos, which is not an unusual scenario, but it was the first server like this that I was a part of so it has a special place in my heart.
